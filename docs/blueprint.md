# **App Name**: Voicepool

## Core Features:

- Google Authentication: Secure user login using Google accounts.
- User Profile Creation: Automatically create user profiles in Firestore upon first login.
- Real-time Audio Rooms: Allow users to join and participate in real-time audio discussion rooms using push-to-talk.
- Voice Recording: Record voice segments during the audio discussions.
- Noise Cancellation: Automatically remove background noise from recorded voice segments.
- Audio Transcription: Transcribe recorded audio segments into text.
- Session Summarization: Summarize the entire audio session using the Gemini model.
- Artifact Generation: Generate a JSON artifact containing metadata, transcript, and summary of the session.
- Artifact Storage: Store the generated JSON artifact in each user's profile.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to reflect trust and reliability.
- Background color: Light gray (#ECEFF1) to ensure readability and a clean interface.
- Accent color: Vibrant orange (#FF9800) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif, providing a modern and neutral look.
- Use minimalist and clear icons for navigation and actions.
- Design a clean and intuitive layout that focuses on audio controls and real-time chat.
- Subtle animations to indicate activity, like voice recording or new messages.
